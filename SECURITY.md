# Security Policy

## Reporting a Vulnerability

Please do not post API keys, tokens, private endpoint details, or sensitive
JSON reports in public issues.

If you need to report a security issue, use GitHub private vulnerability
reporting if available, or contact the maintainer through GitHub.

## Sensitive Data

AI Ping tries to avoid adding full API keys or Authorization headers to reports.
Users should still review all command output and JSON reports before sharing
them publicly.

For compatibility issues, remove secrets before posting commands, logs, or
reports.
